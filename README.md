## Description
This is a simple **Hangman game** built using **HTML, CSS, and JavaScript**.  
The player guesses letters to find the hidden word within a limited number of attempts.  
It is a beginner-friendly project to understand DOM manipulation, events, and basic game logic.

## Features
- Interactive letter guessing
- Dynamic word display
- Limited attempts logic
- Visual feedback for correct and wrong guesses
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
